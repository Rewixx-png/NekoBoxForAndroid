## Поддерживаемые протоколы прокси

* SOCKS (4/4a/5)
* HTTP(S)
* SSH
* Shadowsocks
* VMess
* Trojan
* VLESS
* AnyTLS
* ShadowTLS
* TUIC
* Hysteria 1/2
* WireGuard
* Trojan-Go (trojan-go-plugin)
* NaïveProxy (naive-plugin)
* Mieru (mieru-plugin)

Пожалуйста, перейдите [сюда](https://matsuridayo.github.io/nb4a-plugin/), чтобы скачать плагины для полной поддержки прокси.

## Поддерживаемые форматы подписок

* Некоторые широко используемые форматы, такие как Shadowsocks, ClashMeta и v2rayN
* Исходящие соединения sing-box

Поддерживается только обработка исходящих соединений (узлов). Правила маршрутизации и другие параметры игнорируются.
